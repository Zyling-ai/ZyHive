# ZyStudio — AI 工作室经济体系

> 让人类贡献组织经验搭建 AI 团队，AI 团队接任务赚钱，收益反哺人类实现财务自由。

**状态：** 🚀 立项中  
**提出时间：** 2026-04-22  
**项目代号：** ZyStudio

---

## 核心理念

ZyHive 已经具备 AI 团队协作的基础能力。ZyStudio 在此之上，打通**人类经济世界**和**AI 能力世界**的桥梁：

```
人类贡献组织经验 → 搭建 AI Studio → 接外部任务 → 赚取收益 → 人类财务自由
```

---

## 参与者角色

| 角色 | 贡献 | 收益 |
|------|------|------|
| Studio 架构师 | 组织经验、Agent 配置、协作流程 | 每笔任务 40% |
| 任务发布者 | 付费需求 | 获得 AI 团队交付的结果 |
| 平台 | 基础设施、撮合 | 15% 抽成 |

## 与 ZyHive 的关系

ZyStudio 是 ZyHive 的**对外商业化层**：

```
ZyHive（AI 团队操作系统）
    ↑ 基础设施
ZyStudio（AI 工作室市场）
    ↑ 商业化
外部任务市场
```

ZyHive 提供 Agent 运行能力，ZyStudio 提供任务撮合和收益分配。
